🔥 新手增肌：上下半身分化菜單

一週四練的最佳規劃：週一與週四專攻「上半身」，週二與週五強攻「下半身」。
請選擇一個能在最後兩下感到吃力、但動作標準的重量。組間休息建議 60-90 秒。

1. 坐姿胸推機

胸大肌/三頭

🎯 動作重點 (4組 x 8-12下)

  • 座椅高度： 把手對齊胸部中段（乳頭連線）。
  • 肩胛骨固定： 挺胸、肩膀往後收下壓，**絕不聳肩**。
  • 關節不鎖死： 往前推時吐氣，手肘推到底保持微彎。

2. 坐姿滑輪下拉

背闊肌/二頭

🎯 動作重點 (4組 x 8-12下)

  • 握距： 雙手握在比肩膀略寬之處（桿子彎曲處）。
  • 發力方式： 挺胸收腹，想像用**「手肘」往下帶**，拉到鎖骨至上胸位置。
  • 控制離心： 慢慢放回重量，保持背部張力。

3. 坐姿划船機

背部厚度

🎯 動作重點 (3組 x 8-12下)

  • 軀幹穩定： 雙腳踩穩，背部保持平直不拱背。
  • 啟動順序： 先將「肩胛骨往後夾」，再順勢把把手拉向肚臍。
  • 避免代償： 不要利用腰部過度後仰來甩動借力。

4. 坐姿肩推機

三角肌(肩膀)

🎯 動作重點 (3組 x 8-12下)

  • 手肘位置： 上推時手肘不要全開，稍微**朝前約45度角**，保護肩關節。
  • 核心收緊： 腰部不要過度頂出懸空。
  • 活動範圍： 下放至把手與下巴齊平即可再次上推。

5. 啞鈴二頭彎舉

手臂前側

🎯 動作重點 (3組 x 10-15下)

  • 大臂固定： 上臂緊貼身體兩側，舉起時**大臂絕對不能前後晃動**。
  • 頂峰收縮： 最高點稍微停頓擠壓二頭肌。
  • 緩慢下放： 花 2-3 秒慢慢放下，感受肌肉張力。

6. 滑輪三頭下壓

手臂後側

🎯 動作重點 (3組 x 10-15下)

  • 身體姿勢： 身體微向前傾，核心收緊站穩。
  • 大臂固定： 上臂緊貼身體側邊並保持固定不動。
  • 下壓到底： 往下壓到手臂幾乎伸直，回放時手肘大約呈 90 度即可。

1. 腿部推舉機

大腿前側/臀部

🎯 動作重點 (4組 x 8-12下)

  • 腳掌位置： 雙腳踩在踏板中央，寬度約與肩同寬。
  • 膝蓋方向： 下放時膝蓋朝腳尖方向，**絕對不能往內夾**。
  • 關節不鎖死： 往上推到頂端時膝蓋保持微彎，保護關節。

2. 坐姿腿伸直

大腿前側(股四頭)

🎯 動作重點 (3組 x 10-15下)

  • 器材設定： 背部貼緊椅背，膝關節對齊旋轉軸心，墊子卡在腳踝上方。
  • 頂峰收縮： 往上踢直，在最高點停頓一秒感受強烈收縮。
  • 離心控制： 放下的時候要慢，抵抗重量，不要讓機器「碰」出聲音。

3. 機器腿彎舉

大腿後側(股二頭)

🎯 動作重點 (3組 x 10-15下)

  • 固定身體： 確保大腿上方護墊壓緊，或俯臥時骨盆緊貼墊子不拱腰。
  • 發力重點： 想像用「腳跟」往臀部方向勾到底，感受後側緊繃。
  • 避免代償： 若重量太重會用腰部甩動，請適度減輕重量。

4. 站姿墊腳

小腿肌群

🎯 動作重點 (3組 x 15-20下)

  • 活動範圍： 找台階踩穩腳掌前半部，腳跟懸空。
  • 完整拉伸： 將腳跟盡可能往下壓，讓小腿肌完全拉長。
  • 全力頂起： 用小腿力量將身體墊高到最頂點，停頓一下再慢慢下放。

5. 核心：棒式

核心/腹橫肌

🎯 動作重點 (3組 x 30-60秒)

  • 身體姿勢： 手肘撐在肩膀正下方，頭背臀到腳跟保持一直線。
  • 核心發力： 腹部往內收緊出力（像肚子要被打一拳時的感覺）。
  • 避免塌腰： 若感覺腰痠代表臀部掉下去了，請稍微抬高屁股或先休息！
